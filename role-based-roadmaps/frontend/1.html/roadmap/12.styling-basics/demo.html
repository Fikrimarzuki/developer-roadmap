Demo: Styling Basics

1. Inline CSS

Teks ini di-style dengan Inline CSS (style langsung di tag).

2. Internal CSS (<style> di <head>)

Elemen ini memakai Internal CSS lewat class .internal-box.

3. External CSS (file .css terpisah)

Elemen ini memakai class .external-box. Kalau kamu buat file styles.css, style dari file itu akan diterapkan di sini.

Coba buat file styles.css di folder yang sama dan isi seperti ini: 

/* styles.css */
.external-box {
  background: #fee2e2;
  border: 2px solid #dc2626;
  color: #991b1b;
}
