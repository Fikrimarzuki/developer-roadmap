# Prequesite

## Basic Knowledge Terminal
- cd folder

## Install Python
- Download di web resmi [link](https://www.python.org/downloads/)
- Cek sudah terinstall

Windows
```bash
python --version
```
atau
```bash
py --version
```
Mac/Linux
```bash
python3 --version
```

## Install code editor
- Contoh: VS Code. Extension di VS Code: Python (by Microsoft), Pylance

## Install pip
- Biasanya sudah terinstall
- pip merupakan package manager seperti npm.
- Cek sudah terinstall
```bash
pip --version
```
Jika belum ada, bisa jadi Python PATH belum ter-setup.

## Virtual Environment (Optional tapi sangat direkomendasikan nanti)
- Environment project untuk python agar dependency tidak campur.
- Kalau di Node.js seperti node_modules + package.json
- Mengaktifkan venv

Windows
```bash
python -m venv venv
venv\Scripts\activate
```
Mac/Linux
```bash
python3 -m venv venv
source venv/bin/activate
```
Kalau sudah aktif akan ada `(venv)` di terminal

## Versi Utama Python
- Python 2.x (deprecated)
- Python 3.x (modern)
- Run python
```bash
python file.py
```
```bash
python3 file.py
```
