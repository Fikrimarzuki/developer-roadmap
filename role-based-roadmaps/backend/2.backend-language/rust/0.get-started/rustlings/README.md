# Rustlings

## Requirements
Make sure Rust installed
```
rustc --version
cargo --version
```
if not installed
```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Run Rustlings
Start rustings
```
rustlings
```

## Run Specific Exercise
cargo run --bin <exercise_name>

example:
```
cargo run --bin variables1
```
Notes:
- exercise_name can be found on cargo.toml


## Setup Rustlings in your device
If you want to start rustlings in your device just follow the quick start from this [link](https://rustlings.rust-lang.org/)
- Installation
  ```
  cargo install rustlings
  ```
- Initialization
  ```
  rustlings init
  ```
- Moving into new directory
  ```
  cd rustlings
  ```
- Starting rustlings
  ```
  rustlings
  ```
