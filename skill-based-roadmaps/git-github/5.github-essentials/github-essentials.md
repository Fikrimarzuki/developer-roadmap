- Password

## Creating Repositories
repository adalah folder project online.

cara buat repo
- klik **New Repository**
- Isi nama repo
- Tambahkan deskripsi (jika ada)
- Pilih **Public**(bisa dilihat semua orang) / **Private**(tidak bisa dilihat semua orang)
- Create

contoh untuk nama repository
```
todo-app
portfolio-website
rest-api-node
```
nama jangan tidak jelas seperti
```
project1
test
newrepo
```
