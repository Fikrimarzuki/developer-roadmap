# Setup

## JavaScript di Browser
Salah satu cara menjalankan JavaScript adalah langsung melalui browser.

Beberapa caranya adalah bisa menggunakan:
- **Console** di browser  
- Website online playground seperti **JSFiddle**, **CodePen**, dll.

### Cara membuka Console di browser
1. Buka halaman website apa saja di browser.
2. Klik kanan dan pilih **Inspect**.
3. Buka tab **Console**.
4. Atau gunakan shortcut:
    - **Windows/Linux:** `Ctrl + Shift + I` lalu pilih tab **Console`
    - **Mac:** `Cmd + Option + I`

Di sini bisa langsung mengetik kode JavaScript dan menjalankannya.

## JavaScript di Lokal (Komputer Sendiri)

### 1. Install Browser & Text Editor
- Browser bisa apa saja: **Chrome**, **Edge**, **Firefox**, dll.
- Text editor yang umum dipakai: **Visual Studio Code (VS Code)**.


### 2. Install Node.js
Agar JavaScript bisa dijalankan di komputer (bukan hanya di browser) perlu menginstall **Node.js**.

#### Langkah-langkah:
1. Download Node.js.
2. Install Node.js.
3. Setelah selesai, buka terminal:
   - **Command Prompt**
   - **PowerShell**
   - **Git Bash**
4. Ketik perintah berikut untuk mengecek instalasi:
    ```bash
    node -v
    npm -v
    ```
