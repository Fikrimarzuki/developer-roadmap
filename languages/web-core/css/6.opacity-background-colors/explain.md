# OPACITY, BACKGROUND AND COLORS


## 1. Opacity
mengatur tingkat transparansi elemen
```css
opacity: 0.5;
```
- nilainya antara 0 - 1
- semakin rendah semakin transparan.
- mempengaruhi semua isi elemen, termasuk teks dan gambar



## 2. Background
memberi warna, gambar, atau efek visual pada elemen
### Background Color
```css
background-color: blue;
```

### Background Image
```css
background-image: url("image.jpg");
```
```css
background-size: cover;
background-repeat: no-repeat;
```

### Background Gradient
```css
background: linear-gradient(red, yellow);
```

### Background Position
```css
background-position: center;
```

### Background Attachment
```css
background-attachment: fixed;
```



## 3. Colors
### RGB
```css
color: rgb(255, 0, 0);
```

### HSL
```css
color: hsl(200, 50%, 50%);
```

### HEX
```css
color: #ff0000;
```

### Named Colors
```css
color: blue;
```

### Alpha Colors
transparansi warna saja
```css
color: rgba(255, 0, 0, 0.5);
color: hsla(200, 50%, 50%, 0.4);
```
Opacity berbeda dengan rgba:
- opacity mempengaruhi seluruh elemen
- rgba hanya warna saja
