# DISPLAY


## 1. Inline
```css
display: inline;
```
Ciri:
- Tidak mulai baris baru
- Lebar & tinggi tidak bisa diatur
- Contoh default: `span`, `a`



## 2.Block
```css
display: block;
```
Ciri:
- Selalu mulai baris baru
- Bisa atur width/height
- Mengisi lebar penuh parent
- Contoh default: `div`, `p`, `h1`



## 3. Inline-Block
```css
display: inline-block;
```
Gabungan inline + block:
- Tidak turun baris
- Bisa atur width/height



## 4. None
```css
display: none;
```
Elemen:
- Hilang dari tampilan
- Tidak mengambil ruang layout



## 5. Visibility
```css
visibility: hidden;
```
Elemen:
- Tidak terlihat
- Tapi masih mengambil ruang
