This page is for practicing common HTML text tags and seeing how they look.

2. Paragraph (<p>), Line Break (<br>), Horizontal Rule (<hr>)

Paragraphs are created with the <p> tag. Browsers automatically add some spacing before and after a paragraph.

Sometimes you want a line break
inside the same paragraph without starting a new paragraph. That’s what the <br> tag does.

3. Bold, Strong, Italic, Emphasis, Mark

<b> makes text bold but doesn’t add meaning; it’s just visual.

<strong> makes text bold with semantic importance, meaning “this is important”.

<i> makes text italic but is usually just visual styling.

<em> emphasizes text and is usually read with stress by screen readers.

You can also highlight text with <mark> to show something important or selected.

4. Subscript (<sub>) and Superscript (<sup>)

Subscript is text that sits slightly below the baseline.
Example: H2O

Superscript is text that sits slightly above the baseline.
Example: x2 + y2 = r2

5. Preformatted Text (<pre>)

The <pre> tag keeps spaces, line breaks, and uses a monospace font. Useful for code blocks or text that must keep its formatting: 

Line 1: this   has   extra   spaces
Line 2:    and indentation
Line 3: <pre> shows text exactly as written.
