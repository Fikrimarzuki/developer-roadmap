Demo: Including JavaScript

1. Inline JavaScript

2. Internal JavaScript (<script> di HTML)

3. External JavaScript (file .js terpisah)

Isi file app.js seperti ini:

/* app.js */
document.addEventListener("DOMContentLoaded", () => {
  const btn = document.getElementById("external-btn");
  const msg = document.getElementById("external-message");

  btn.addEventListener("click", () => {
    msg.textContent = "External JS: button clicked!";
  });
});
